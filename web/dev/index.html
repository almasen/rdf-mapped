RDFmapped Dev

Welcome to the Developer Site of the RDFmapped website!

You can find module documentation, live test coverage and many more here along with references to the official GitHub repository.

Contributing

We warmly welcome all kinds of contributions from the tiniest UI tweaks through bug reports to expanding our collection of featured LinkedIn Learning courses & videos.

Contributing code

If you are interested in contributing code, please have look at the planned features and open issues on the GitHub repository page or create your own feature request / bugfix.

Contributing content

If you are interested in contributing data and have a dataset that you would like to submit for us to review, please use the contact form on the official website and select the "Dataset to be imported" option as your reason. For submitting a single LinkedIn Learning course or video, please use the Learning Object submission form.

Report a vulnerability

For security related issues, such as reporting a vulnerability, please consult the security policy.

With any questions you may have, do feel free to contact us.

If you wish to get back to the main site, navigate to https://rdfmapped.com.
